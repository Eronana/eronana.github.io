CF534C Polycarpus' Dice

#题意
有n个骰子,地i个骰子可投掷出1-di
现在给出n个骰子的值的和A
问每个骰子不可能的值有多少个
2 3
2 3
两个骰子,总和是3
第一个1的话,第二个就2
第一个2的话,第二个就1
所以第一个骰子没有不可能的值
第二个同上,可为1,可为2,但是不可为3,于是不可能的值的个数为1
最终输出0 1

##解题思路
对所有di求和的S
很显然,可投掷出的总值的区间为[n,S]
对于第i个骰子,若掷出1,则最大值为S-di+1
若掷出di,则最小值为n+di-1
也就是说,若a>S-di+1,则必须依靠第i个骰子来调节才可能使总和为a
换句话说,就是第i个骰子就会出现不可能的值
于是很容易得出,不可能取值的个数为a-(S-di+1)
对于a<n+di-1的情况也如此
总结一下
若a>S-di+1,则有a-(S-di+1)个值不可取
若aa<n+di-1,则有n+di-1-a个值不可取
ps:2*10^5个10^6相加会爆int,所以要用long long

CF538C Tourist's Notes

##题意
爬山日志记录了每天的高度:第i天的高度为$h_i$
并且由于选择了十分平坦的道路,两天的高度只差不会大于1
爬山结束后回到家,发现日志丢失了很多,只剩下一部分
现在你需要根据这些日志找出可能到达的最高的高度
如果日志记录有误,输出IMPOSSIBLE

##解题思路
这题比较偏数学,解法是列两个不等式
若i和j是残缺日志中相邻的两天
设第k天是第i天到第j天中最高的那天,则有
$
\begin{cases}
h_k \leq h_i+(k-i) \
h_k \leq h_j+(j-k)
\end{cases}
$
解得
$h_k \leq \frac{h_i+h_j+j-i}{2}$
有了公式就简单了,开头和结尾单独计算,然后对所有i,j,算出$h_k$取最大值即可
最后是判断IMPOSSIBLE,如果$\left|h_j-h_i|\right>j-i$就输出IMPOSSIBLE

保证$1 \leq n \leq 10000, 1 \leq ai \leq 10000$ 对于任意 $1 \leq i \leq n$, 且对于任意$1 \leq i, j \leq n(i!=j)$，满足$ai$ != $aj$

若存在,x[j]就是f(i),累加到ans即可

HDU5187 zhx's contest

##题意
n个数,从1到n
问能做成多少个漂亮的序列
漂亮的序列需要满足下面两个条件:
a1...ai是单调递增或单调递减
ai...an是单调递增或单调递减
答案很大,需要对p取模

##解题思路
简单思考后不难想到,要达成漂亮的序列,关键在于最大的数n
n的位置确定了,序列的个数就确定了
若n在第i位,则从a1..ai满足递增个数为C(n-1,i-1),满足递减的个数同样为C(n-1,i-1)
由于最大的是n,所以一定存在由剩下的数组成的满足递增或递减的序列
于是答案便是$2\sum{i=2}^{n}C{n-1}^{i-1}$
不过直接求的话,复杂度相当高,于是需要对该式进行化简
令m=n-1,则上式的
$2\sum_{i=1}^{m}Cm^i$
根据二项式定理可得
$2^n=\sum{i=0}^{n}Cn^i$
于是$2\sum{i=1}^{m}C_m^i=2*2^m-2=2^n-2$
剩下的就是编码实现了
求幂自然是快速幂算法,不过需要注意一点便是由于要对p取模
所以公式上要做一些改动:$(2^n-2+p)\ mod\ p$
还有就是,由于数据量略大,进行快速幂时做乘法运算可能会超出long long的范围
所以做乘法需要用快速乘法算法
快速乘法其实并不快,只是运用同快速幂相同的思想用加法做乘法,于是可以避免超范围

CF520C DNA Alignment

##题意
假定字符串s和t的长度都是n,则函数h(s,t)表示位置和字符都相同的个数
然后函数p(s,t)的定义是

shit(s,i)是将s循环移动i位.例如
ρ("AGC", "CGT") = 
h("AGC", "CGT") + h("AGC", "GTC") + h("AGC", "TCG") + 
h("GCA", "CGT") + h("GCA", "GTC") + h("GCA", "TCG") + 
h("CAG", "CGT") + h("CAG", "GTC") + h("CAG", "TCG") = 
1 + 1 + 0 + 0 + 1 + 1 + 1 + 0 + 1 = 6
给出一个字符串s,求有多少个字符串t,满足p(s,t)最大
答案很大,需要对10^9+7取模

##解题思路
首先观察函数p(s,t)
对于每个s,t都移动过不同的顺序,其值之和就是s和t中相同的字符的个数
而s也要移动n次,所以p(s,t)的实质是s和t中相同的字符数*n
有了这个结论就好说了
对于给定的s,使p(s,t)最大,只需让t全部为s中出现最多的字符即可
若s中,有m个字符出现的次数相同且最大
则对于t中的每个字符,为m个字符中的任意一个,对结果都不产生影响
于是t总共有m^n种情况,本题已破
总结一下
计算出s中出现最多的字符的有多少种,记为m
然后使用快速幂计算pow(m,n)%1000000007即可

CF519C A and B and Team Training

##题意
有小白和大牛两种生物
一组为三人,必须这两种
于是有两种分组方法
小白,大牛,大牛
小白,小白,大牛
给出大牛的数量n和小白的数量m,问组多能分多少组

##解题思路
首先很容易想到大概是(n+m)/3
但是如果人数差距太大,这个公式就是错的
比如n=1,m=1000.很明显只能组成一组
也就是如果其中一种人数大于另一种人数的二倍,那么多出来的人数将毫无意义
于是如果n>m则交换n和m
然后判断m是否>n*2,若大于n*2则令m=n*2
最后输出(n+m)/3即可

CF519B A and B and Compilation Errors

##题意
共有n个数
第一行给出这n个数
然后从n个数中去除一个
第二行给出去除后的n-1个数
然后再去除一个
第三行给出去两个数后的n-2个数
找出去除的两个数

##解题思路
求出第一行的和s1,第二行的和s2,第三行的和s3
ans1=s1-s2
ans2=s2-s3
看下数据范围,直接求和会溢出的
不过似乎由于补码的特性,即使溢出了,结果仍然正确
为了保险起见,我们可以用异或
a^a=0,也即是两个相等的数进行异或运算,其结果为0
而0^a=a,也就是和0进行异或后,值不发生改变
简单来说就是两个相同的数,通过异或后会抵消
于是第一行异或的值为x1,第二行异或值为x2,第三行异或值为x3
ans1=x1^x2
ans2=x2^x3

CF515C Drazil and Factorial

##题意
F(123456)=1!*2!*3!*4!*5!*6!
给出一个整数a,求满足下面条件的最大的x
F(a)=F(x)
十进制的x的每一位数字,都不包含1和0

##解题思路
首先我们的任务是干掉0和1,并让F的值不变
关于0和1,直接拿掉即可
因为0!和1!都是1,做乘法,结果不会有任何变化
不过我们需要求的是最大的x
首先我们已经知道,0和1可以忽略,于是来看看其他的数
4!=4*3!
并且4=2!*2!
于是4!可以写成2!*2!*3!
其他个数如法炮制即可
0和1无视
2!无法分解,3!无法分解,5!无法分解,7!无法分解
4!=3!*2!*2!
6!=3!*5!
8!=7!*2!*2!*2!
9!=7!*3!*3!*2!
于是本题已破,打个表扫一遍,最后再降序排序,答案就出来了
