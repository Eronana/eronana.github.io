POJ3169 Layout

##题意
N个奶牛按照从1到N的顺序排队,并且奶牛们会有这两种要求
A奶牛和B奶牛的距离不能超过D
A奶牛和B奶牛的距离不能小于D
求第一只奶牛和第N只奶牛的最大距离
若无法达成奶牛的要求,输出-1
如果1和N的距离可以无限远,输出-2

##解题思路
很明显的差分约束系统
求最大距离,就要求最短路
根据两个条件,添加相符的边即可
不过本题有一个蕴含条件就是相邻的两只奶牛的距离要>0,因为是按照从1到N的顺序排队的
而差分约束系统除了有确定的解外,还有其他两种情况
无解和任意多的解,也就是对应这本题的另外两种输出
当无解时,会出现负环,当有无限个解时,最短路为INF

POJ3159 Candies

##题意
给n个熊孩子发糖,m个条件:
每个条件是:第Ai个熊孩子要求第Bi个熊孩子的糖不能比他多Ci个
在保证满足所有条件的前提下,问第n个孩子比第1个孩子做多能多多少个糖果

##解题思路
很典型的差分约束系统
解决差分约束问题的方式是把不等式转换为图
最短路中,若存在权值为w路径的u->v
设dist[x]为源点s到x的最短路长度
若s到v的最短路经过u->v这条边,则dist[v]=dist[u]+w
若不经过这条边,则dist[v]<dist[u]+w
于是有dist[v]<=dist[u]+w
回到本题上,把熊孩子看成点,用d[x]表示第x个孩子得到的糖果数量
每个条件就可以写成d[B]-d[A]<=C
整理一下,得d[b]<=d[A]+C
形式上和上面最短路的不等式一样
于是把每个条件看做边,熊孩子看做点,构图,求最短路
即可得到熊孩子和源点熊孩子的最大差
为什么是最大差呢,再观察不等式d[b]<=d[A]+C
什么时候d[b]最大呢,很显然,是取等号的时候
而最短路所走的边,都是取等号的时候,所以结果得到的是最大差
虽然对于一般的查分约束系统来说,都存在负权,要用SPFA
而本体边权都为正,所以可以使用更快的Dijkstra

POJ3169 Layout

奶牛们按降序排队,也有可能多个奶牛在同一排
然后有一些奶牛关系好,他们不能超过一定的距离
有一些奶牛关系不好,他们必须间隔一定的距离
然后你的任务是确定在这些条件下,1号奶牛和N号奶牛的最小距离是多少

由于是降序的,所以建图时注意编号的顺序

P tip

d(B)-d(A)<=-X
V tip
d(B)-d(A)<=-1

POJ1201 Intervals

略有难度的差分约束题
给出n个如下条件
在[ai,bi]区间内至少选择ci个元素
求出总共最少选择多少个元素可满足约束条件
继续前缀和思想
Ti表示从头[0,i]区间内选择多少个元素
于是条件变为
$$T_bi-T_{ai-1}\geq ci$$
不过题目范围是0 <= ai <= bi <= 50000
0-1就变成-1了,于是这里需要做一下修改
另Ti表示[0,i-1]内选择的数量,则条件变为
$$T_{bi+1}-T_ai\geq ci$$
由于是大于等于,所以就变为求最长路径了
不够目前条件还不够,还需要增加条件
$$0\leq T_{i+1}-T_i\leq 1$$
这个不等式拆成两个,总共3个条件
然后建图,SPFA即可
其实次题用Dijkstra也可以
因为如果把区间内所有元素都选择的话,一定满足约束条件
也就是说,至少会有这1种解,即不存在负权(对应最长路是正权)回路

POJ1364 King

入门级的差分约束
既然给的是和的形式,就用前缀和来破
另$$T_i=\sum_{0\leq j\leq i}$$
则$$\sum_{a\leq j\leq b}< k$$可变换为$$T_b-T_{a-1}< k$$
还差个等号,由于是整数,直接k-1即可,于是
$$T_b-T_{a-1}\leq k-1$$
建好图,套上SPFA即可
